Development rules
=================

Editing
-------
* 100 caracters per line
* 4 spaces for indent
* camel case method
* feature name in lower case
* doc and comment files in MarkDown

Comments
--------
* multiligne comment above complex function
    /*
    *
    *
    */

Code formating
--------------
* java brace style

    methode{
        function{
        }
    }

* <?php ?>
* xhtml (lower cased balises)
* css

    div#example_01 {
        padding: 3px;
        margin: 33px;
        width: 333px;
    }
